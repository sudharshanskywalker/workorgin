# Worko - Local Service Professional Platform (Python Flask Version)

Worko is a web application connecting local service workers (plumbers, electricians, etc.) with customers.

## features

- **Find Professionals**: Search and filter local skilled workers.
- **Worker Profiles**: Detailed profiles with ratings, reviews, and skills.
- **Booking Flow**: (Mock) Interface to book services.
- **Join as Pro**: Registration form for service providers.

## Prerequisites

You need **Python 3.x** installed on your system.

## Getting Started

1. **Install Dependencies**
   Open a terminal in this folder and run:
   ```bash
   pip install -r requirements.txt
   ```

2. **Run the Application**
   Start the Flask server:
   ```bash
   python app.py
   ```

3. **Open the App**
   Open your browser and navigate to `http://127.0.0.1:5000`.

## Technologies

- Python (Flask)
- Jinja2 Templates
- Tailwind CSS (via CDN)
- Lucide Icons (via CDN)
